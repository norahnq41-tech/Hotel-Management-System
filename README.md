# Hotel Management System

A comprehensive Hotel Management System developed using Java, Spring Boot, and JavaFX. This project is designed to automate and streamline hospitality operations, providing a robust backend for managing guests, room reservations, and staff workflows.

## Key Features
- **Guest Management:** Efficiently record and manage guest information.
- **Room Booking System:** Seamlessly handle bookings for various room types (Single, Double, Suite).
- **Accommodation & Restaurant Services:** Integrated management for room amenities and restaurant reservations.
- **Data Persistence:** Utilizes robust File I/O operations (MyFileReader/MyFileWriter) to maintain system data integrity.
- **Modern UI:** Built with JavaFX to ensure a user-friendly and interactive interface.

## Tech Stack
- **Language:** Java
- **Frameworks:** Spring Boot, JavaFX
- **Concepts:** Object-Oriented Programming (OOP), Data Handling, System Architecture.
